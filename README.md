🏡 Real Estate Analytics Dashboard (Power BI)

📌 Project Overview
This project focuses on analyzing real estate data using Power BI. The goal was to transform raw and unclean data into meaningful insights for better decision-making.

🧹 Data Cleaning
- Handled missing values  
- Removed duplicates  
- Corrected data types  
- Managed outliers  
- Standardized data  
- Created new features  

📊 Dashboard Features
- Summary Insights  
- Location Analysis  
- School Impact Analysis  
- Feature-Based Pricing Insights  

💡 Key Insights
- Location plays a major role in pricing  
- Property features increase value significantly  
- School ratings impact demand  

🎯 Tools Used
- Power BI  
- Data Cleaning Techniques
